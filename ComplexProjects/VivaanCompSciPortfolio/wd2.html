Unit Two: Web Development II

In this unit, we learned how to style in CSS. We learned that HTML is for the structure to the page and CSS is used to style the page (give the look and feel to the page). Below are the CSS selectors that we learned in this unit and my favorite project.

CSS Selectors:

Format: { tag/.class/#id: property; }; <> with 'tag selector="name"' inside angle brackets

CSS Selector Selector Explaination
Tag This is used to select all elements with a specific type of tag.
Class This is used to select all elements with a specific assigned class.
ID This is used to select one element with the specific assigned id.
